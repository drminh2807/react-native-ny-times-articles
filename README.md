# react-native-ny-times-articles

## Install Homebrew
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Install Node
```
brew install node
```

## Install Watchman
```
brew install watchman
```

## Install Yarn
```
brew install yarn
```

## Install Expo CLI
```
npm install -g expo-cli
```

## Install node_modules
```
yarn
```

## Run Project
```
yarn start
```

## Test
```
yarn test
```
